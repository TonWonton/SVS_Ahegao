# SVS_Ahegao

## Description
Ahegao (アヘ顔) plugin / MOD for `Samabake Scramble` / `サマバケ！すくらんぶる`. Allows you to set 3 different faces for 3 different states for both female and male characters. The different conditions for each state can also be configured.

## Prerequisites
- [BepInEx Unity Il2Cpp](https://github.com/BepInEx/BepInEx)
- [BepisPlugins for SVS](https://github.com/IllusionMods/BepisPlugins)
- [BepInEx.ConfigurationManager Il2Cpp](https://github.com/BepInEx/BepInEx.ConfigurationManager)

#### OR
- [SVS-HF_Patch](https://github.com/ManlyMarco/SVS-HF_Patch)

## Installation
1. Install prerequisites
2. Download from releases
3. Extract into game folder (.DLL file should be in `/GAME_FOLDER/BepInEx/plugins/`)

## Features
- Set custom faces for 3 different states
  - Orgasm
  - Faintness
  - Faintness when above a certain speed
- Male ahegao
  - Toggle male ahegao on/off
- Change eye type, blinking, open/close amount, eye roll/cross, eyebrows, etc.
- Change mouth type, open/close amount, etc.
- Set blush and tear level
- Change amount of orgasms needed for faintness state
- Change minimum speed required for the different states

## Notes
- Not compatible with `SVS_HSceneAddons`

## Known issues
- Orgasm ahegao will trigger even if it shouldn't for some if the gauge is above 80% during climax
